# muslibger

Your personal music library manager

## Installation
*soon*

## Some instructions

You need cargo to build/test project yourself

### Build:
```bash
cargo build

# or with optimizations
cargo build --release
```
binary will be located under ./target/debug or ./target/release

### Direct run:
```bash
cargo run -- [args]
```

### Run tests:
```bash
 cargo test -- --show-output
```
